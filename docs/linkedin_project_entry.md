# LinkedIn Project Entry

## Project Title

From Text to Visuals: AI Pipeline for Immersive Storytelling

## Short Description

Researched and prototyped an AI pipeline that transforms literary text into visual scene prompts using NLP, genre classification, prompt engineering, and image generation.

## Project Description

Wrote a 31-page research paper exploring how AI can make literature more immersive by converting short stories into visual representations. Using a dataset of Edgar Allan Poe short stories, I designed a pipeline involving text preprocessing, tokenization, genre classification, and prompt generation for AI image models.

The project tested how GPT-style language models and DALL-E-style image generation could translate narrative tone, setting, characters, and imagery into visual outputs. A major focus was not just creating images, but understanding the challenges of narrative consistency, prompt quality, rate limits, accessibility, and preserving the meaning of literary text.
